A note on classes

Throughout this document, you will see the class attribute set on various elements. The class attribute, like the id attribute, lets us set a label on elements we can then refer to in our css. However, unlike the id, the class attribute can be used multiple times in the same document and is usually used to denote a category of elements. For example, cats are a category of animals (class), but my cat has the specific name "Ripley" (id). We select classes in by using a . and the name of the class.

Width, Margins, Padding and Borders

We can set width, margin, padding, and border properties on block elements. These values work differently, sometimes not at all, for inline elements. The way these values control how block elements take up space on the page is called the box model. The basic concept is simple. The width property sets the width of the element. The border sets the visible border of the element (if any). The margin sets the space between the border and the next element, and the padding sets the space between the border and the element's content. We'll see each of these in turn now.

Width

The width property lets us specify the width of a given element. Widths can be specified in three ways:

  • As a percentage of the element's container (more on this in a minute).
  • As a pixel value
  • As an em value

Just as with font size, pixels are a absolute measurement value whereas ems and percentages are relative. We'll discuss the merits of relative vs absolute measurements later on. Containers

When one block level element contains another, the larger element is called the container. The body element is the container of all other elements on the page. By default, block elements take up 100% of their container. When we set a percentage width, the element will take up that percentage of its container. Thus, the actual width of the element will depend on the width of the container itself! In the example below, the div with the class "demoContainer" is the container. Margin

The margin property allows us to set space between the border of an element and its neighbors. The margin property can take one, two, or four values. If given one value, this value with set the margin for all four sides of the element. If given two values the first value sets the vertical margin and the second sets the horizontal. We'll deal with four later (it's rarely used).

Padding

The padding property lets us set space between the border of an element and its content. Like the margin, it can take one, two or four values. A one value padding is applied to all four sides. For a two value padding the first value sets vertical padding and the second, horizontal.
